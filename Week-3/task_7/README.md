# Завдання 7. 
Створити програму, поділену на три модулі: main.py, ui.py та logic.py. У
модулі ui.py описати клас головного вікна з елементами інтерфейсу, у logic.py —
функції для обробки даних, у main.py — головну частину програми з імпортом
модулів і запуском інтерфейсу. Переконатися, що інтерфейс і логіка працюють
незалежно один від одного.
