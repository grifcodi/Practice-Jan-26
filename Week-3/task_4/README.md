# Завдання 4. 
Створити програму «Калькулятор», яка дозволяє вводити два числа та
виконувати основні арифметичні операції: додавання, віднімання, множення та
ділення. Результат обчислень має відображатись у полі Label. Забезпечити обробку
помилок — зокрема, випадок ділення на нуль або уведення нечислових значень.
